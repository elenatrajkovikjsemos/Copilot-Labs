Lab 1 - Copilot in Word

This lab guides you through a Word on the web exercise where Copilot helps turn a short sales draft into a polished, business-ready document for a software solutions company.

Note: Use Copilot in Word (Web) to edit content directly and ask questions about the document. The examples below show how to build the document, then review and refine it using Copilot's content assistance.

🧩 Copilot in Word Exercise – Sales Company

Scenario

You work in a sales company that offers software solutions. You start from a short draft and use Copilot to develop it into a more complete, business-ready document.

How to use Copilot in Word (Web)

  • Edit with Copilot
    Use this to create and improve content directly in the document.
    👉 Select text → give instruction → Copilot updates the document
  • Ask Copilot
    Use this to analyze, review, and give suggestions about the document.
    👉 Works as discussion only (does not change the document)

Starting Document

Our company provides software solutions to help businesses manage their sales processes more efficiently. We focus on improving customer relationships, tracking opportunities, and increasing revenue.

Recently, we have seen increased interest from clients in automation and data-driven decision making.

Part 1: Edit with Copilot (build the document)

  1. Task 1 – Expand the content
    Select the full document and instruct Copilot:
    Expand this into a more complete description of a sales company offering software solutions, including what we offer, who our customers are, and how we deliver value.
  2. Task 2 – Add missing section
    Place your cursor at the end of the document and ask Copilot to:
    Add a new paragraph describing our sales approach and how we engage with potential customers.
  3. Task 3 – Improve quality
    Select the whole document and instruct Copilot:
    Make this more professional, clear, and suitable for a business audience. Add title, headings and all appropriate formatting.

Part 2: Ask Copilot (analyze and refine)

Once the document is improved, use Copilot to review it and identify further enhancements.

  1. Prompt 1
    Ask Copilot:
    What is still missing from this document if it should present a strong sales offering?
  2. Prompt 2
    Ask Copilot:
    How could this document be improved to better appeal to potential clients?
  3. Prompt 3
    Ask Copilot:
    Does the document clearly explain the company’s value? If not, what should be improved?

Use the feedback from Copilot to refine your document further, adding clarity, stronger value statements, and a more persuasive sales tone.
