🧩 Copilot Agents Exercise – Specialized Assistants

Scenario

You are working in a sales and strategy team and need support for:

  • analyzing internal sales data
  • researching market trends and positioning

Instead of writing prompts from scratch, you will use specialized agents to support your work.

Agents are specialized AI assistants that help automate tasks, analyze data, and provide structured insights.

How to use Copilot Agents

🤖 Use prebuilt agents

Use this to:

  • work with specialized assistants (Analyst, Researcher)
  • get structured, role-based answers

🛠️ Create your own agent

Use this to:

  • define a specific role and behavior
  • reuse the same logic across scenarios
  • enrich it with knowledge sources

👉 You describe the agent → add instructions → add knowledge → test it

🔹 Part 1: Use built-in agents

✅ Analyst Agent

What it does

The Analyst agent helps you:

  • analyze datasets (like Excel)
  • identify trends and anomalies
  • summarize performance and metrics

👉 Use your Excel file: Contoso Chai Tea market trends

Prompt 1

Analyze this dataset and identify key sales trends across the year, including growth patterns and seasonal changes.

Prompt 2

Identify any anomalies or unusual values in the data and explain what might be causing them.

✅ Researcher Agent

What it does

The Researcher agent helps you:

  • gather insights from internal and external sources
  • analyze market trends
  • produce structured research output

Prompt 3

Create a short research report on current trends in software sales for business clients, including key strategies, challenges, and future outlook.

🔹 Part 2: Create your own agent (Agent Builder)

Agent Builder allows you to create an agent directly in Copilot using natural language, without coding.

Step 1: Describe your agent

Example description:

A Sales Strategy Assistant that helps analyze sales performance, interpret trends, and suggest actions to improve business results.

Step 2: Define instructions

Example instructions:

Provide clear and structured analysis of sales data.
Highlight key trends, risks, and opportunities.
Give concise and actionable recommendations.
Use a professional and business-focused tone.

Step 3: Add knowledge

Enhance your agent by adding knowledge sources such as:

  • SharePoint documents (internal sales materials, reports)
  • OneDrive files
  • Public websites with market insights

Step 4: Test your agent

Example prompts:

Analyze our sales data and provide the most important insights.
Based on current market trends, what should be our top sales priorities?
