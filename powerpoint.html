Lab 2 - Build a presentation with Copilot in PowerPoint

This lab helps you turn your completed Word document into an internal presentation using Copilot in PowerPoint. You'll quickly create slides, improve messaging, and review your deck for a business audience.

Note: Copilot in PowerPoint can both create and modify slides directly, and it can also provide feedback without changing the deck. Use the two modes together for faster iteration.

🧩 Copilot in PowerPoint Exercise – Sales Company

Scenario

You now need to turn your completed Word document into a presentation for internal use. The goal is to quickly create and refine a presentation using Copilot in PowerPoint.

How to use Copilot in PowerPoint (Web)

  • Create / Edit with Copilot
    Use this to:
    • generate slides from content
    • rewrite or improve slide text
    • add speaker notes
    • suggest visual improvements
    👉 This modifies or creates slides
  • Ask Copilot
    Use this to:
    • review the presentation
    • suggest improvements
    • give feedback and ideas
    👉 This does not directly modify slides

Part 1: Create the presentation

Open PowerPoint and start a new presentation.

  1. Task 1 – Generate slides from the Word content
    Use Copilot to create a presentation based on the content of this document.
  2. Review the generated slides.

Part 2: Edit with Copilot (improve the presentation)

  1. Task 1 – Refine messaging
    Select a slide that describes the company and ask Copilot:
    Rewrite this slide to better communicate the company’s value to customers.
  2. Task 2 – Add speaker notes
    Ask Copilot to add speaker notes with more detailed explanations for each slide.
    Add speaker notes with more detailed explanation for each slide.
  3. Task 3 – Add simple animations
    Ask Copilot to add subtle animations.
    Add subtle animations to make this presentation easier to present.

Part 3: Ask Copilot (review and feedback)

Now evaluate the presentation using Copilot's review capabilities.

  1. Prompt 1
    Ask Copilot: Does this presentation clearly communicate the company’s offering and value?
  2. Prompt 2
    Ask Copilot: Which slides may need more explanation when presenting? What should I focus on when presenting these slides?
  3. Prompt 3
    Ask Copilot: What could make this presentation more convincing for business decision-makers?

Use Copilot's feedback to make final refinements and ensure the presentation is clear, persuasive, and easy for internal audiences to understand.
