🧩 Copilot in Excel

Scenario

You are analyzing sales performance for Contoso’s Chai product line.

You will use Copilot in Excel to:

  • enrich your dataset with calculations
  • highlight important patterns
  • build quick reports
  • analyze data and support decisions

Dataset source file: Contoso Chai Tea market trends.xlsx

Note: You must download the source file from the link and upload it to your OneDrive.

How to use Copilot in Excel

  • ✏️ Edit with Copilot
    Use this to create and update data directly in your worksheet.
    👉 Add columns → apply formulas → build visuals → Copilot updates the file
  • 💬 Ask Copilot
    Use this to analyze, interpret, and get insights from your data.
    👉 Works as discussion only (does not change the data unless you ask it to)

🔹 Part 1: Edit your data

Prompt 1

Add a new column that calculates the percentage of Artisanal Chai Sales compared to Total Chai Sales. Apply the formula to all rows.

Prompt 2

Add a column that calculates the month-over-month growth rate of Total Chai Sales as a percentage.

Prompt 3

Apply conditional formatting to highlight the top 3 months with the highest Total Chai Sales and the bottom 3 months with the lowest values.

Prompt 4

Create a pivot table showing Total, Artisanal, and Premade Chai Sales by month. Then create a chart to visualize the trends.

🔹 Part 2: Ask Copilot (analysis & decisions)

Prompt 5

Analyze this dataset and summarize the key trends in sales performance across the year.

Prompt 6

Identify any unusual or inconsistent values in the dataset and explain why they might be important.

Prompt 7

Compare the performance of Artisanal vs Premade Chai Sales and explain the main differences.

Prompt 8

Based on this data, what recommendations would you give to improve sales performance?
